Projeto: Dados dos Atletas

Este projeto foi desenvolvido como parte do curso DevStart para praticar Programação Orientada a Objetos (POO) em JavaScript.  
O objetivo é criar uma classe chamada Atleta que organiza e processa informações de atletas, como nome, idade, peso, altura e notas de desempenho.



 Funcionalidades

A classe Atleta possui os seguintes métodos:

 Métodos de cálculo
- calculaCategoria() = Retorna a categoria do atleta com base na idade.  
  - *Infantil*: 9 a 11 anos  
  - *Juvenil*: 12 a 13 anos  
  - *Intermediário*: 14 a 15 anos  
  - *Adulto*: 16 a 30 anos  
  - *Sem categoria*: fora das faixas acima  

- calculaIMC() = Calcula o Índice de Massa Corporal (IMC) do atleta usando a fórmula:
- IMC = peso / (altura * altura)
- 
- calcular Media Valida  → Calcula a média das notas do atleta, desconsiderando a maior e a menor nota.
